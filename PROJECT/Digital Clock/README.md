## Digital Clock using Python and Tkinter

This script create a digital clock as per the system's current time.

## Library Used

<li>tkinter</li>
<li>time</li>

## To install required external modules

Run :
pip install tkinter

## How to run the script

Execute python3 digital_clock.py
